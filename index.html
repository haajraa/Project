Project¶

Given a list of Twitter user handles U1 to U10 create a dictionary of words used across all of their tweets and output the size of this dictionary with the following properties. You will also need to output the vector of top used words for each user.

  • The tweets need to be fetched from Twitter as much as possible for each user. For testing purposes, use 10 text files called U1.txt to U10.txt. Each tweet is on a single line in each file.
  • The dictionary should not contain stop words (e.g. "the", "at", "as", etc.). The list of stop words is attached in a text file called "stopwords.txt" as a comma separated list.
  • The words in the dictionary should not contain any letters with more than 2 repetitions. All these repetitions will be replaced with a single character. For example "angryyyyy" will be stored in the dictionary as "angry".
  • For each user top word vector, the output should be an ordered list of top 10 words. The primary ordering criteria should be the frequency of the word in the user's tweets and the secondary sort would be alphabetical ordering.

Once you have the dictionary for each user, write a method that when called and given a twitter handle, would be able to rank the other 9 users based on their similarity to the given user.

Defining the Methods to measure the Cosine similarity of Vectors

In [15]:
import numpy as np
def cos_sim(a,b):
    dot_product = np.dot(a,b)
    norm_a = np.linalg.norm(a)
    norm_b = np.linalg.norm(b)
    return dot_product/(norm_a * norm_b)

def calculateSimilarity(dict1,dict2):
    all_words_list = []
    for key in dict1:
        all_words_list.append(key)
    for key in dict2:
        all_words_list.append(key) 
    #to remove the duplicates from the all word list
    all_words_lst = list(set(all_words_list))
    
    all_words_list_size = len(all_words_list)
    #create 2 vectors/1D arrays of all_words_list_size
    v1 = np.zeros(all_words_list_size, dtype=np.int)
    v2 = np.zeros(all_words_list_size, dtype=np.int)
    i = 0
    for(key) in all_words_list:
        v1[i] = dict1.get(key,0)
        v2[i] = dict2.get(key,0)
        i = i +1
    return cos_sim(v1,v2)
